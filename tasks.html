Problem Statement

Напишіть функцію treeMinValue, яка приймає корінь бінарного дерева. Функція має повернути мінімальне значення, що міститься в дереві. Можна припустити, що дерево не порожнє.

Best for Interviews

const treeMinValue = (root) => {

if (root === null) return Infinity;

const leftMin = treeMinValue(root.left);

const rightMin = treeMinValue(root.right);

return Math.min(root.val, leftMin, rightMin);

};

Important Note about BFS

"Хоча ітеративний BFS працює, у JS shift() займає O(n), тому для істинної продуктивності O(n) варто використовувати власну чергу або стек для DFS."
